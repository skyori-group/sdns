О системе

Система дистанционного электронного голосования (СДЭГ) позволяет организовать дистанционное голосование депутатов по вопросам повестки в режиме онлайн. Депутат, находясь вне зала заседаний, имеет доступ к голосованию аналогично системе зала заседаний. Депутат видит рассматриваемые вопросы и может голосовать по ним.
Параллельно должна быть развернута ВКС, по ней проходят обсуждения.

Как проходит дистанционное заседание

  1. Администратор заносит в систему информацию о составе участников, вносит повестку
  2. Депутаты через браузер заходят на страницу СДЭГ и авторизуются с помощью логина и пароля (при необходимости можно настроить подтверждение по смс)
  3. По ВКС председатель объявляет, какой вопрос находится на рассмотрении. Оператор отмечает в списке вопросов повестки заседания текущий рассматриваемый. Депутат видит этот вопрос на экране приложения СДЭГ
  4. При объявлении голосования, оператор запускает голосование по текущему вопросу. Депутаты видят на экране варианты: «за», «против», «воздержался» и таймер обратного отсчета
  5. После завершения времени голосования вычисляется итог: принято/ не принято решение. На экранах участников выводится информация о количественных итогах.
  6. После завершения заседания оператор может выгрузить результаты дистанционного голосования депутатов по вопросам заседания.

эффекты внедрения

  • возможность проводить полноценные заседания в период карантина и сложной эпидемиологической обстановки
  • возможность оперативно голосовать по вопросам федеральной повестки в промежутке между плановыми заседаниями

Характеристки

  • СДЭГ реализована в виде web-приложения. Работает в браузерах на ПК мобильных устройствах

требования к программному обеспечению

Сервер базы данных: 2-ядерный процессор, объем оперативной памяти 8-16 Гб
  • СУБД: MS SQL Server 2008 R2 (и выше) или Postgre SQL 9.6 (и выше)
Серверная ОС: 2-ядерный процессор, объем оперативной памяти 4-8 Гб
  • Windows 2012R2 Server и выше, CentOS 7 и выше, Debian 9 и выше, Ubuntu версий 19.10, 18.04, 16.04
Клиентские рабочие места: наличие любого из перечисленных браузеров
  • Яндекс.Браузер версии 19 и выше
  • Google Chrome версии 70 и выше
  • Mozilla Firefox версии 70 и выше
